lệnh start dư án 

Chạy Back-end:
Mở terminal tại thư mục backEnd và chạy lệnh:

npm i
# Import database bằng lệnh sau để tránh lỗi font tiếng Việt:
mysql -u root -p --default-character-set=utf8mb4 < backEnd/database/schema.sql
# Hoặc trên Windows PowerShell:
# cmd.exe /c "mysql -u root -p --default-character-set=utf8mb4 < backEnd/database/schema.sql"
npm run dev


Chạy Front-end:
Mở terminal tại thư mục frontEnd và chạy lệnh:

npm i
npm run dev
